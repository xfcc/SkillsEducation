# SkillsEducation：专业教育通用架构 (PoC)

> 泛知识类产品兜售“信息焦虑”，而专业教育产品交付“核心胜任力”。

**SkillsEducation** 是一个概念验证小程序项目。本项目旨在探索并确立“专业教育（Professional Education）”品类的产品架构范式。

本项目以高门槛的心脏电生理（EP）医生培训为业务沙盘，尝试将庞杂的医疗资讯、学术课程与硬核的手术模拟工具，重构为一套高信噪比、SaaS化的学习工具。这套底层架构同样适用于工程、高端制造、法律、投资等一切依赖“刻意练习”的专业领域。

## 🎯 核心设计哲学：知与行的物理隔离

当前市面上绝大多数的教育 App 都是“内容大卖场”。它们试图用大量的内容信息来占据用户时间。

SkillsEducation 引入了顶级工具型产品的心智折叠法则。我们认为，**所有硬核专业技能的习得，必然且只能划分为两个物理与认知上高度独立的阶段——理论（Input）与演练（Output）。**

* **理论 (知)**：系统向用户提供标准答案。目的是构建心智模型。
* **演练 (行)**：用户向系统输出决策动作。目的是暴露技能盲区与建立肌肉记忆。

基于这一哲学，我们在产品架构上对“知”与“行”进行了冷酷且彻底的界定与隔离。
